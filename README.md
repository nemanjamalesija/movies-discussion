SOCIALIS - social network

FEATURES:

- Sign up
- Log in
- Send friend requests
- Accept friend requests
- Delete friend from friends list
- Posts feed (home page)
- Search users
- Look up users' individual profiles
- Post
- Like post
- Unlike post
- Delete post
- Comment
- Edit comment
- Delete comment

TECHNOLOGIES USED:  
Vue 3, Pina, Zod, Tailwind css, Express, Mongo DB.
